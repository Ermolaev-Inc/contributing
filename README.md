## Commits 
Pattern: `<type>(<scope?>): <description>` <br>
Types:
- feat - feature
- fix - fix
- test - test
- ref - refactoring
- ui - ui
- docs - docs
- v - version
- o - other

Examples:
- `feat(Button): create`
- `v: bump to 2.0.0`
