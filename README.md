## Commits 
Pattern: `<type>(<scope?>): <description>` <br>
Types:
- f - feature
- b - bug
- t - test
- r - refactoring
- d - docs
- v - version
- o - other

Examples:
- `f(Button): create`
- `v: bump to 2.0.0`
